# Binary Addition


Given two binary strings `a` and `b`, return  *their sum as a binary string* .

**Example 1:**

<pre><strong>Input:</strong> a = "11", b = "1"
<strong>Output:</strong> "100"
</pre>

**Example 2:**

<pre><strong>Input:</strong> a = "1010", b = "1011"
<strong>Output:</strong> "10101"
</pre>
